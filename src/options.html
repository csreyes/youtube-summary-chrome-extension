YouTube AI Summarizer Options

API Settings

Your API key is stored locally on your device and is not sent anywhere besides OpenRouter.

About OpenRouter: This extension uses OpenRouter to access various AI models for summarization and chat.

To get an API key:

  1. Go to openrouter.ai/keys
  2. Sign up or log in
  3. Create a new API key
  4. Copy and paste the key here

OpenRouter offers free credits for testing, and affordable pay-as-you-go for continued use.

See the OpenRouter documentation for available models.

Chat & Summary Settings

Available variables:

  • {{transcript}} - The video transcript
  • {{title}} - The video title
  • {{channel}} - The channel name
  • {{url}} - The video URL
0 (More focused) 0.7 1 (More creative)

Controls the randomness of the AI's responses. Lower values produce more focused and deterministic responses.
